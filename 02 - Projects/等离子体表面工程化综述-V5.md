## 自绘图片说明

### Fig 4: 两性离子代表单体结构 (SBMA / CBMA)
- 色块式化学结构图（避免复杂 sketcher 依赖）
- 主链：CH₂ → C(CH₃) → COO
- SBMA: N⁺(CH₃)₂ + SO₃⁻
- CBMA: N⁺(CH₃)₂ + COO⁻

### Fig 5: 表面修饰方法对比
- 三列：PECVD/eATRP、Graft-to、Dopamine
- 每列：流程步骤 + pros + cons（对齐修正）
- 右侧：性能对比表（厚度、抗蛋白效果、耐久性）
- 数据来源：文献综合

### Fig 6: 致病级联与干预机制
- 六步链条：材料植入 → 免疫炎症 → 蛋白吸附 → 细胞黏附 → 组织纤维化 → 临床失败
- 底部绿色框：两性离子水合层干预机制，90-99% 蛋白吸附下降，切断步骤 2-6
