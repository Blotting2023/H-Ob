### 1.2 核心产品线详解

#### （1）ALLEVYN 系列 — 战略核心（利润产品）

| 维度 | 内容 |
|------|------|
| **技术类型** | 聚氨酯泡沫敷料（Siltex 微孔界面层 + 高吸收芯层 + 防水背膜） |
| **临床定位** | 慢性创面（糖尿病足溃疡DFU、压力性损伤PI、静脉性腿部溃疡VLU）的一线标准护理 |
| **目标创面** | 中等-高渗出的浅表至中度深度创面 |
| **技术层级** | 成熟技术，全球占有率高，大规格/含银版本为高毛利品种 |
| **与金华泽为技术重叠度** | **低**。金华泽为的两性离子涂层是表面化学创新，而ALLEVYN是物理结构（泡沫+吸水层）为主。金华泽为的技术可被视为**叠加层**，理论上可以负载在泡沫敷料基材上，形成互补而非替代。 |

#### （2）PICO 系列 — 战略增长引擎（高壁垒利润产品）

| 维度 | 内容 |
|------|------|
| **技术类型** | 一次性负压伤口治疗（sNPWT），便携电泵 + 敷料套件 |
| **临床定位** | 术后切口管理、高阶慢性创面，替代传统墙式NPWT |
| **目标创面** | 手术切口（降低SSI风险）、慢性创面需加速愈合 |
| **技术壁垒** | 高。泵体微型化设计、气密性敷料贴合技术、电池续航管理均为S&N专利簇 |
| **与金华泽为技术重叠度** | **几乎无重叠**。PICO属于**主动治疗**（机械力干预），金华泽为属**被动功能材料**（表面化学改性的防污/促愈）。两赛道逻辑完全不同。 |

#### （3）ACTICOAT 系列 — 高毛利抗菌线（利润产品）

| 维度 | 内容 |
|------|------|
| **技术类型** | 纳米晶银（Nanocrystalline Silver）涂层技术 |
| **临床定位** | 感染创面/高感染风险的慢性创面 |
| **目标创面** | DFU合并感染、烧伤创面、术后感染创面 |
| **壁垒** | 纳米晶银制造工艺专利（S&N独有），7天持续释放 |
| **与金华泽为技术重叠度** | **中低**。抗菌维度有交叉——金华泽为的两性离子涂层本身也有一定的防生物膜/抗菌性能，但作用机制根本不同（ACTICOAT=银离子杀菌，金华泽为=表面抗蛋白吸附→破坏生物膜起始附着）。金华泽为技术理论上可对银过敏患者提供非银替代方案。 |

#### （4）REGRAVEX — 唯一平台级生物药（绝对壁垒）

| 维度 | 内容 |
|------|------|
| **技术类型** | 重组人血小板衍生生长因子(rhPDGF-BB)局部凝胶 |
| **临床定位** | 神经性糖尿病足溃疡（FDA Label） |
| **壁垒** | 唯一FDA批准的局部生长因子创面药物；CMC工艺壁垒极高 |
| **商业现状** | 价格昂贵（~$500+/支），医保覆盖有限，面临仿生竞争（赫赛克/其他Biosimilar） |
| **与金华泽为关系** | 分类不同（药物 vs 器械），金华泽为无直接竞争 |

#### （5）OPSITE 系列 — 流量产品

| 维度 | 内容 |
|------|------|
| **技术类型** | 聚氨酯透明薄膜/涂胶层 |
| **临床定位** | 术后防水、IV置管固定、浅表创面覆盖 |
| **目标创面** | 手术切口、穿刺点 |
| **策略** | 低单价、高周转、大渠道走量 |
| **与金华泽为关系** | 金华泽为技术可做OPSITE的功能升级替代品——同样是薄膜型敷料，但多了防污/促愈涂层 |

### 1.3 产品矩阵策略（BCG+Price定位）

| 角色 | 产品 | 毛利率估算 | 核心策略 |
|------|------|-----------|---------|
| **流量产品** (低价高频) | OPSITE系列、CURITY纱布 | ~40-50% | 院线基础耗材走量，维系渠道关系和品牌曝光 |

## 2. 注册准入策略

### 2.1 FDA Product Codes 映射

| Product Code | 描述 | S&N对应产品线 |
|-------------|------|-------------|
| **FRO** | Dressing, Wound, Drug | ACTICOAT（含银药物敷料） |
| **KGN** | Dressing, Wound, Hydrogel | INTRASITE Gel、REGRAVEX可归属此路径 |
| **MGP** | Dressing, Wound, Occlusive | OPSITE系列（透明薄膜封闭敷料） |
| **MGR** | Dressing, Wound, Hydrocolloid | ALLEVYN系列部分型号可归类此处 |
| **KGX** | Powered Suction Pump, Portable, Wound | PICO（便携电动负压吸引泵） |
| **FRO/MGP交叉** | 含银封闭敷料 | ALLEVYN Ag（同时涉及银药物成分和封闭结构） |

> 注册路径选择逻辑：S&N通常选择 **510(k) premarket notification** 路径（非PMA），因为大部分产品属于"实质性等同"（Substantial Equivalence）于已上市产品。REGRAVEX因属生长因子药物，走的是 **BLA (Biologics License Application)** 路径。

**在华注册策略判断**：
- S&N中国区早期走的是**"从上到下"**路径——先拿III类注册证保高壁垒产品（含银/含药敷料），再用II类放大通路产品铺量
- 近年NMPA对含银敷料监管趋严（需做生物学评价+药代动力学），S&N面临注册维护成本上升
- PICO在中国市场的注册策略是**"器械拆分"**——泵体（II类）+ 敷料耗材（II类/III类）分别注册

### 2.4 注册优先级逻辑

S&N的全球注册策略呈现清晰的三层分法：

```
第一优先级（同步中美欧）
    └── PICO sNPWT（高壁垒、高单价、竞争少）
    └── ALLEVYN Life/Ag（全球支柱产品，任何市场缺失都是重大损失）

第二优先级（先欧美后新兴）
    └── ACTICOAT（含银敷料注册成本高，优先在欧美做高认知）
    └── REGRAVEX（药品路径，各市场独立审批）

第三优先级（区域化/本地化）
    └── OPSITE基本型（本地化生产或OEM竞品替代）
    └── 基础纱布类（新兴市场本地供应）
```

### 3.2 临床指南中的产品推荐

|---------|---------|------------------|

> **策略解码**：S&N的学术营销路径为：**临床研究(RCT) → 系统综述/荟萃分析 → 指南引用 → 政策准入(医保/集采豁免)**。PICO是这一路径的完美执行案例——从WHIST试验到NICE MTG，完成了从证据到支付的全链条。金华泽为若走B2B模式，需复制此路径但可用AI/RWD（真实世界数据）加速。

### 3.4 学术营销战术拆解

S&N的学术营销形成标准化闭环：

```
基础研究（材料物理/生物学）
    ↓
研究者发起研究(IIT)：覆盖真实世界数据/病例系列
    ↓
公司支持RCT：多中心、高证据等级（如WHIST试验~£1.5M投资）
    ↓
系统综述/荟萃分析（通常第三方学术团队主导）
    ↓
指南提交（NICE/IWGDF/WOCN审阅委员会——S&N可提供证据包）
    ↓
指南正式推荐 → 支付方谈判（医保/DRG覆盖）→ 院内销量暴增
```

**关键启示**：S&N每年在学术营销上的投入估算占AWM营收的 **6-10%**（约$80-135M），包括研究者发起研究、医学教育、会议赞助、出版物推广等。这是金华泽为类创业公司**无法直接复制的资源投入**。但金华泽为可聚焦一个适应症（如DFU/导管感染）做**深度破局**，而非大而全。

**整体判断**：S&N中国区创面护理业务是 **"代理为主+直销为辅"** 的混合模式。

- **高值产品（PICO、ACTICOAT）**：采用 **直销+学术跟台** 模式，S&N中国有专门的AWM销售团队（约200-300人[待核实]），覆盖全国Top 200医院
- **中低端产品（OPSITE、ALLEVYN基础型）**：通过 **区域代理商/配送商**（国药、华润、上药等）分发，S&N团队主要做KOL管理
- **院外市场**：非核心战场，由代理商自发铺货，S&N很少直接投入

**关键变化趋势（2023-2026）**：
1. 集采压力下，S&N中国在考虑**本土化生产**降成本
2. PICO和含银敷料尚在集采保护期，但预计2026-2027年面临新一轮价格谈判
3. S&N正在加大**县域医院**覆盖——低线城市创面中心建设是国家政策方向

**关键交叉点**：

2. **含银敷料替代（竞争度：中）**：金华泽为技术若证明非银途径具有同等或更好的抗生物膜效果，则可在银过敏患者（占人口~10-15%）和"无银化"趋势中替代ACTICOAT。

3. **泡沫敷料（竞争度：低-中）**：金华泽为可以开发"两性离子涂层泡沫敷料"作为ALLEVYN的功能升级替代品，但需解决涂层在三维多孔结构中的均匀性问题。

4. **NPWT（竞争度：无）**：PICO与金华泽为完全不在一个赛道上。

### 5.4 潜在合作场景

虽然金华泽为与S&N存在竞争关系，但合作空间同样存在：
