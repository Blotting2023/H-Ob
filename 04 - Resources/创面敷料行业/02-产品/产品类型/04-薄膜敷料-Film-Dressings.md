# 薄膜敷料 (Film Dressings)

## 市场规模
- 全球：~$1.5B (2026) [待核实]
- CAGR：~4.0%
- 中国市场：~¥18亿 [待核实]

## 结构与成分
- **基材**：聚氨酯（PU）透明薄膜
- **活性成分**：通常无活性成分（部分含碘/银抗菌涂层）
- **典型结构**：单层半透明聚氨酯薄膜 + 丙烯酸酯粘合剂涂层（或硅酮粘性边框）

## 作用机制
- 半透性屏障：允许氧气和水蒸气通过（WVTR可控），阻止细菌/病毒和液体进入
- 保持湿润环境：减少经皮水分蒸发（TEWL），维持伤口湿润
- 透明观察：无需移除即可观察伤口变化
- 顺应性：超薄薄膜贴合身体轮廓，薄至 0.02-0.04mm

## 临床适应症
- **主要**：**静脉留置针固定**、低渗出/无渗出浅表伤口（擦伤、I度烧伤）、术后切口/缝合线保护、供皮区薄层覆盖、水胶体/泡沫敷料的二层固定
- **禁忌**：高渗出伤口、感染伤口、需清创的全层伤口、潮湿/浸渍周围皮肤

## 优点
- 超薄透明，可直接观察伤口
- 防水、防菌、透气
- 自粘性好，无需额外固定
- 舒适/隐蔽，患者接受度高
- 配合中心静脉置管/留置针固定标准

## 缺点
- 几乎无吸收能力（不适合中高渗液）
- 剥离时容易损伤脆弱皮肤（尤其老年/新生儿）
- 边缘可能卷曲脱落
- 积聚渗液时气泡/掀起风险
- 水分滞留可能导致皮肤浸渍

## 注册路径
- **NMPA**：II类医疗器械
- **FDA**：510(k) Class II
- **CE**：MDR Class IIa

## 技术趋势
- 硅酮粘性替代丙烯酸酯（减少皮肤损伤）
- 高透气（WVTR > 3000 g/m²/24h）
- 含抗菌涂层薄膜
- 高顺应性超薄（<15μm）
- 可视化窗口+吸液垫复合
- 药物缓释薄膜

## 与金华泽为关系
- **竞争/互补**：互补。薄膜敷料功能定位在固定和保护无渗出伤口/穿刺点，金华泽涂层敷料面向慢性难愈合创面，场景差异大
- **替代可能**：极低。两类产品解决完全不同的问题
