# 负压引流敷料 (NPWT — Negative Pressure Wound Therapy Dressings)

## 市场规模
- 全球：~$0.8B (2026，仅耗材部分) [待核实]
- CAGR：~6.5%（设备+耗材合计市场约$3.0B，此处仅计敷料耗材部分）
- 中国市场：~¥10亿（耗材部分）[待核实]

## 结构与成分
- **基材**：聚氨酯（PU）泡沫 / 聚乙烯醇（PVA）海绵
- **活性成分**：无（传统NPWT），新型NPWT含抗菌银/凝胶涂层
- **典型结构**：
  - **填充海绵**：开孔PU泡沫（黑色）或PVA海绵（白色），裁剪填充创面
  - **封闭膜**：透明聚氨酯薄膜（气密性）
  - **引流管/管路**：内置引流管连接负压源
  - **收集罐**：一次性的液体收集容器
  - 部分系统含洗注孔（Instill NPWT）

## 作用机制
- **宏观变形**：负压（-80~-125mmHg）使海绵塌陷收缩，创面机械收缩，减小创面面积
- **微观机械拉伸**：负压对壁面细胞施加周期性张力（microdeformations），促进细胞增殖和血管化
- **渗液清除**：持续/间歇负压吸除过量渗液和感染性液体（减少MMP和炎症介质）
- **水肿减轻**：减少间质水肿，改善局部微循环
- **创面环境稳定**：封闭系统防止外源性感染
- **肉芽组织形成**：机械刺激+渗液清除双重作用促进肉芽生长

## 临床适应症
- **主要**：**复杂/慢性难愈合创面**（压疮III/IV期、糖尿病足溃疡、静脉淤血溃疡）、**急性创伤创面**、**术后切口**（尤其高感染风险的胸骨/腹部切口）、**植皮前创面准备**、**植皮后固定**、**开腹减压创面**
- **禁忌**：未清创的坏死组织/焦痂、恶性肿瘤伤口、暴露的血管/吻合口、未经控制的活动性出血、肠瘘（无隔离）、未处理的骨髓炎

## 优点
- 显著加速肉芽组织生长
- 有效缩小创面面积
- 降低创面感染率
- 减少换药频率（每48-72h一次）
- 改善植皮成功率
- 减少住院时间和医护工作量

## 缺点
- **设备依赖**（需泵+耗材配套）
- **高成本**（设备+耗材）
- 疼痛（换药时可能明显）
- 活动受限（携带负压泵）
- 漏气/堵管等设备相关问题
- 不适用于浅表低渗出创面
- 噪音（泵运行）

## 注册路径
- **NMPA**：**III类医疗器械**（有源设备+耗材组合）
- **FDA**：Class II 510(k) / Class III PMA（部分）
- **CE**：MDR Class IIb/III

## 技术趋势
- **一次性便携NPWT**（PICO模式，无泵/机械抽吸）
- Instill NPWT：间断灌注抗菌液+负压的清洗模式
- NPWT+滴注（洗注一体化）
- 可穿戴/家用NPWT系统
- NPWT+生物敷料联合（胶原/NPWT复合）
- 无线智能负压监测
- 减少尺寸/重量，改善患者生活质量

## 与金华泽为关系
- **竞争/互补**：**互补**。NPWT针对深部/复杂创面的机械清创和渗液管理，金华泽涂层敷料在NPWT去除后或联合使用时提供抗感染/抗生物膜保护
- **替代可能**：低。NPWT和涂层敷料解决完全不同的创面管理环节；金华泽涂层敷料可作为NPWT停用后的延续治疗
