# 胶原敷料 (Collagen Dressings)

## 市场规模
- 全球：~$0.9B (2026) [待核实]
- CAGR：~7.0%
- 中国市场：~¥10亿 [待核实]

## 结构与成分
- **基材**：I型/III型胶原蛋白（主要来源：牛跟腱、猪皮、鱼皮、重组人胶原）
- **活性成分**：天然胶原蛋白多肽、部分含弹性蛋白、透明质酸、生长因子复合
- **典型结构**：冻干海绵片、纤维无纺布、凝胶、粉末、膜状

## 作用机制
- **细胞外基质支架**：提供ECM结构模板，引导成纤维细胞和内皮细胞迁入
- **促胶原沉积**：降解产物（胶原肽）刺激宿主成纤维细胞合成新胶原
- **趋化作用**：胶原降解产物吸引巨噬细胞和成纤维细胞到创面
- **结合MMP**：降低创面过量基质金属蛋白酶（MMPs）水平（慢性创面MMP失衡的关键修复障碍）
- **提供愈合支架**：多孔结构支持血管化和肉芽组织形成
- **止血**：血小板粘附和聚集（胶原诱导）

## 临床适应症
- **主要**：**慢性难愈合创面**（糖尿病足溃疡、静脉淤血溃疡、压疮III/IV期）、**急性创面**（手术切口、烧伤供皮区、外伤创面）、**营养缺乏伤口**
- **禁忌**：感染未控制伤口、胶原过敏（罕见）、坏死组织未清创的创面（胶原作为细菌底物）

## 优点
- 促进慢性难愈合伤口修复的主动治疗性敷料
- 降低MMP活性，纠正创面愈合微环境失衡
- 提供ECM支架，引导组织再生
- 良好的止血效果
- 非粘性（湿润后软化）
- 可降解吸收

## 缺点
- **成本高**（显著高于传统敷料）
- 需二次固定敷料覆盖
- 不能用于感染未控制伤口（提供细菌营养）
- 冷藏/特定储存条件
- 效果依赖伤口床准备质量
- 对接受数量有要求（常规需3-6周连续使用）

## 注册路径
- **NMPA**：III类医疗器械（植入/组织修复类）
- **FDA**：510(k) Class II / PMA Class III（含活细胞产品）
- **CE**：MDR Class III（组织工程类）

## 技术趋势
- 重组人胶原（无动物源、GMP下游杂质可控）
- 胶原+生长因子复合（PDGF-bFGF-VEGF负载）
- 交联胶原（延长降解时间，增强机械性能）
- 胶原+透明质酸/硫酸软骨素复合
- 3D打印胶原支架（定制化创面修复）
- 脱细胞ECM+自体细胞培养（组织工程皮肤）

## 与金华泽为关系
- **竞争/互补**：互补。胶原敷料核心在"促修复"（ECM支架+MMP调节），金华泽核心在"防感染"（抗蛋白吸附+抗生物膜），是最理想的协同组合
- **替代可能**：低。两个产品针对不同的核心病理环节，建议联合使用
