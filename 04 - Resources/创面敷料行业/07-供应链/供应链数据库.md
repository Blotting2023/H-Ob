## 供应链全景总览

```
上游原料 ──→ 中游制造 ──→ 下游渠道 ──→ 终端用户
   │               │               │
   ├─ 聚合物基材    ├─ 涂布工艺      ├─ 医院（招标/集采）
   ├─ 功能单体      ├─ 接枝/修饰     ├─ 药店零售
   ├─ 抗菌剂        ├─ 灭菌（EtO/    ├─ 电商（国内/海外）
   ├─ 生物材料       │   电子束/γ）  ├─ OEM/ODM代工
   └─ 供应商图谱     ├─ 包装          └─ 海外渠道
                    └─ 设备厂商
```

## 一、上游原料

### 聚合物基材（6类）
| 材料 | 特性 | 应用 |
|------|------|------|
| CMC（羧甲基纤维素钠） | 高吸湿、凝胶化 | 水胶体敷料 |
| PVA（聚乙烯醇） | 成膜、亲水、可降解 | 水凝胶、静电纺丝 |
| PU（聚氨酯） | 弹性、透气、防水 | 薄膜/泡沫敷料 |
| PET（聚对苯二甲酸乙二醇酯） | 高强度、尺寸稳定 | 背衬层、无纺布 |
| PLA（聚乳酸） | 可生物降解 | 可吸收敷料 |
| PLGA（乳酸-羟基乙酸共聚物） | 降解可调、缓释载体 | 载药敷料 |

### 功能单体（4种）
SBMA（抗污渍）、PCBMA（抗菌黏附）、DMAEMA（pH响应）、NIPAM（温敏，LCST~32°C）

### 抗菌剂（5大类）
- 银基：纳米银、磺胺嘧啶银
- 有机：PHMB、聚维酮碘
- 天然：壳聚糖、蜂蜜、植物提取物

### 生物材料（5种）
胶原、透明质酸（HA）、海藻酸钠、明胶、丝素蛋白

### 核心供应商
- **国际**：Merck/Sigma-Aldrich、Evonik（RESOMER®）、BASF、DuPont、Ashland
- **国产**：山东赫达（CMC）、皖维高新（PVA）、万华化学（PU）、华熙生物（HA）、长春圣博玛（PLGA）、烟台赛诺（壳聚糖）、浙江海正（PLA）

## 二、中游制造

### 涂布工艺（6种）
| 工艺 | 精度 | 应用 |
|------|------|------|
| 刮涂 | 10-500 μm | 水胶体、PU膜 |
| 喷涂 | — | 抗菌涂层 |
| 浸涂 | — | 海绵/无纺布载药 |
| 旋涂 | — | 实验室精密涂层 |
| 静电纺丝 | 50 nm-5 μm | 纳米纤维膜 |
| 3D打印 | 高精度 | 个性化定制 |

### 接枝修饰（4种）
等离子体处理、UV接枝、电子束辐照、湿化学修饰

### 灭菌（4种）
| 方法 | 典型参数 | 适用材料 |
|------|----------|----------|
| EtO | 37-55°C, 450-1200 mg/L, 1-6h | 热敏/含药敷料（需7-14天解析） |
| 电子束 | **10-25 kGy**（典型剂量） | 干态敷料、PU/PET膜 |
| γ辐照 | 15-40 kGy | 密封/厚包装产品 |
| 蒸汽 | 121°C/15min 或 134°C/3min | 耐热材料 |

### 包装
灭菌屏障包装（Tyvek® + PET/PE膜）、符合ISO 11607

### 关键设备厂商
- **涂布**：Coatema（德）、Mathis（瑞）、西安航天华阳（中）
- **灭菌**：Sterigenics（美）、IBA（比）、中广核技（中）
- **静电纺丝**：Elmarco（捷）、Inovenso（土）、苏州纳米科技（中）
- **3D打印**：CELLINK/BICO（瑞）、杭州捷诺飞（中）
- **包装**：MULTIVAC（德）、上海迈威（中）

## 三、下游渠道

### 渠道结构
- **医院**（60-70%）：集采影响显著（降幅40-60%），省际联盟推进中
- **药店**（15-20%）：连锁药店（大参林/老百姓/一心堂）+ 区域连锁
- **电商**：
  - 国内：天猫/京东/拼多多/小红书
  - 海外：Amazon / eBay / 独立站
- **OEM/ODM**：振德、稳健、奥美、海氏海诺为全球主要代工厂
- **海外渠道**：FDA（美）、CE MDR（欧）、ANVISA（巴西）
  - 美国GPO：Vizient, Premier
  - 国际品牌合作：ConvaTec, Smith+Nephew, Mölnlycke, 3M

## 四、供应链关键数据

### 成本结构（估计）
| 环节 | 占比 |
|------|------|
| 原料成本 | 20-30% |
| 制造+灭菌 | 20-25% |
| 包装 | 5-10% |
| 渠道/营销 | 30-40% |
| 净利润 | 5-15% (高端 > 低端) |

### 代表产品供应链路径
**水胶体敷料**：CMC（山东赫达）→ 刮涂复合（振德医疗）→ 电子束灭菌（10-25 kGy）→ Tyvek® 包装 → 医院招标/药店零售

**硅酮泡沫敷料**：PU（万华化学/Evonik）→ 发泡成型 + 硅酮涂层（喷涂）→ EtO灭菌 → 吸塑包装 → 医院/Amazon

**纳米银抗菌敷料**：PET无纺布 + AgNP（Merck）→ 浸涂/喷涂 → γ辐照灭菌 → 灭菌袋 → 医院招标（高附加值）

**海藻酸盐敷料**：海藻酸钠（青岛明月）→ 纺丝成网 → γ辐照灭菌 → 纸塑袋 → 医院/药店

## 五、趋势与风险

### 趋势
- 集采推动行业集中度提升
- 创新敷料（智能响应、载药、可穿戴）提高附加值
- 国产替代加速（上游原料 + 中游设备）
- 跨境电商直达终端（DTC模式）
- 环保灭菌（电子束替代EtO趋势）

### 风险
- 集采降幅超预期压缩利润
- 原材料价格波动（CMC/HA/PU）
- 出口认证壁垒（FDA/MDR）
- 灭菌产能瓶颈（电子束加速器投资高）
- 中小OEM厂商同质化竞争
