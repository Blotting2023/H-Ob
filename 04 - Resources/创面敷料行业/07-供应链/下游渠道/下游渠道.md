# 下游渠道

## 一、医院招标（集采影响）

### 1.1 医院终端特点
- 占创面敷料终端市场的 **60-70%**（中国）
- 决策链：医院 → 科室主任（临床需求） → 设备科（采购流程） → 院领导审批
- 准入壁垒：各省市医保耗材目录、医院耗材遴选

### 1.2 集采政策影响
- **国家级集采（带量采购）**：已覆盖高值耗材，低值耗材（含常规敷料）逐步纳入
- **省际联盟集采**：
  - 河南省际联盟（2023）：医用敷料集采，平均降幅约 40-60%
  - 江苏省集采：创面敷料品类集采（2022-2024）
- **影响分析**：
  - 利好：龙头企业（稳健/振德/奥美）中标后量增
  - 利空：中小厂商利润压缩，价格战加剧
  - 趋势：从"高毛利低量"转向"低毛利高量"

### 1.3 医院准入流程
1. 产品注册（NMPA II类/III类）
2. 省级耗材挂网（医保编码）
3. 医院耗材遴选评审（临床+经济性评估）
4. 招标/议价
5. 配送商签约（国药/华润/上药/九州通等）

## 二、药店零售

### 2.1 药店终端特点
- 占终端市场的 **15-20%**
- 产品以OTC类为主：创可贴、普通纱布、水胶体（痘痘贴）、硅酮凝胶（疤痕贴）
- 渠道分类：
  - **连锁药店**：大参林、老百姓、一心堂、益丰、国大药房（全国连锁）
  - **区域连锁**：海王星辰、漱玉平民、健之佳
  - **单体药店**：社区/乡镇药店

### 2.2 进入药店渠道要求
- 药店进场费、条码费
- 地推团队（覆盖各门店）
- 首推激励/陈列费
- 会员营销合作

## 三、电商渠道

### 3.1 国内电商

#### 天猫/淘宝
- 品牌旗舰店（品牌直营）+ 专营店（经销商）
- 运营成本：佣金 2-5% + 推广费（直通车/钻展）
- 品类热门：痘痘贴、疤痕贴、硅酮凝胶、水胶体

#### 京东
- 京东自营（入仓）+ 品牌旗舰店
- 京东物流优势
- 医疗器械资质要求更严格

#### 拼多多
- 低价策略，适合基础敷料（创可贴/纱布）
- 毛利率较低但量大盘大

#### 小红书（种草引流）
- 核心品类：痘痘贴、疤痕修复、医美术后护理
- KOL/KOC种草 → 站外转化

### 3.2 海外电商

#### Amazon
- 美国站、欧洲站、日本站
- 要求：FDA 510(k) / CE MDR 认证
- 运营：FBA（亚马逊物流）为主
- 热门品类：硅胶疤痕贴、水胶体痘痘贴、泡沫敷料

#### eBay / 速卖通（AliExpress）
- 适合跨境中小卖家
- 价格竞争激烈

#### 独立站（Shopify / Magento）
- 适合品牌化运营（如 Medvanced, Welly）
- 需要SEO + 社交媒体引流（Facebook/Instagram/TikTok）

## 四、OEM/ODM 代工

### 4.1 代工模式
| 模式 | 定义 | 典型角色 |
|------|------|----------|
| OEM | 品牌提供设计，代工厂生产 | 振德、稳健、奥美 |
| ODM | 代工厂提供设计+生产 | 全棉时代、海氏海诺 |
| OBM | 代工厂自建品牌 | 云南白药、可孚 |

| 厂商 | 地点 | 产能 | 核心品类 |
|------|------|------|----------|

| 奥美医疗 | 湖北枝江 | 全球最大医用敷料OEM之一 | 纱布、无纺布、手术巾 |

### 4.3 代工合作要点
- 保密协议（NDA）
- 质量协议（ISO 13485、CE、FDA）
- 模具开发费、起订量（MOQ）
- 交货周期（通常 30-60 天）
- 品控标准（AQL 抽样）

## 五、海外渠道

### 5.1 目标市场
| 地区 | 市场规模 | 准入要求 | 渠道特点 |
|------|----------|----------|----------|
| 美国 | 最大（全球 ~40%） | FDA 510(k) / PMA | GPO 集团采购、分销商（Medline, McKesson, Cardinal Health） |
| 欧盟 | 第二 | CE MDR（2024年强制） | 各国招标 + 经销商模式 |
| 东南亚 | 快速成长 | 东盟各国注册 | 进口分销 + 当地代理商 |
| 中东 & 非洲 | 新兴 | 各国注册（依赖CE） | 政府采购 + 分销商 |
| 拉美 | 新兴 | ANVISA（巴西）/ COFEPRIS（墨西哥） | 进口分销 |

### 5.2 海外渠道结构
- **制造商** → **进口商/品牌商** → **分销商** → **医院/药店** → **终端患者**
- **GPO（集团采购组织）**：美国市场重要渠道，如 Vizient, Premier, HealthTrust
- **直接合作**：大型跨国品牌（3M/KCI/Smith+Nephew/ConvaTec）
- **电商直达**：Amazon 跨境直营

### 5.4 海外拓展路径建议
1. 获取目标市场注册（FDA/MDR/东盟）
2. 参加国际展会（MEDICA/杜塞尔多夫、IDEA/迈阿密）
3. 建立经销网络（区域独家代理商）
4. 设立海外仓（Amazon FBA / 第三方仓）
5. 考虑海外子公司或合资公司
