## 二、分类对照表

### 按产品类型分类对比

| 产品类型 | NMPA分类 | FDA Product Code | CE MDR分类 | 备注 |
|----------|----------|-----------------|-------------|------|
| 普通纱布/棉垫 | II类 | FRO | Class I | 最基础的敷料 |
| 透明薄膜敷料 | II类 | FRO | Class I | 聚氨酯薄膜 |
| 水胶体敷料 | II类 | FRO | Class I / IIa | 视创面宣称 |
| 泡沫敷料 | II类 | FRO | Class I / IIa | ⚠️ MDR可能升IIa |
| 藻酸盐敷料 | II类 | KGN | Class I / IIa | 亲水性可吸收 |
| 水凝胶敷料 | II类 | MGP | Class I / IIa | 高含水量 |
| 硅酮敷料 | II类 | FRO | Class I | 接触层 |
| 含银敷料 | II类 | MGR | IIa / IIb | 抗菌功能 |
| 可吸收止血敷料 | III类 | KGN | IIb | 体内停留 |
| 含活细胞敷料 | III类 | 组合产品 | III | 组织工程产品 |
| 含生长因子敷料 | III类 | 组合产品/PMA | III | 生物活性成分 |

### 快速判定要点

- **NMPA**：主要看是否含药理活性成分、是否可吸收、是否含活体细胞
- **FDA**：主要看吸收性、含水量、是否抗菌
- **CE MDR**：主要看是否涉及慢性/深度/大面积创面、是否含药、是否可吸收

## 五、注册路线选择速查表

| 企业场景 | NMPA建议路径 | FDA建议路径 | CE MDR建议路径 |
|----------|-------------|-------------|----------------|
| 初创企业/基础敷料 | II类注册（省级局） | 510(k)（FRO/KGN） | Class I自声明 或 IIa+NB |
| 含抗菌剂敷料 | II类注册（同品种评价） | 510(k)（MGR） | IIa/IIb + NB |
| 含药/含活性物质 | III类注册 + 临床试验 | PMA / 组合产品路径 | III类 + NB |
| 已有MDD证书 | 不适用 | 不适用 | MDR转换（NB审核） |
| 已有NMPA证书欲出海 | 基准 | 510(k)（同品对比） | MDR（GSPRs编制） |
| 金华泽式企业 | II类注册 + 临床豁免 | 510(k)（FRO/MGP） | IIa + 尽早签约NB |
