### 1.2 FDA 510(k) 数据库检索式

**官方入口**：https://www.accessdata.fda.gov/scripts/cdrh/cfdocs/cfpmn/pmn.cfm

**手动检索方法**：
1. 进入 [510(k) Premarket Notification Database](https://www.accessdata.fda.gov/scripts/cdrh/cfdocs/cfpmn/pmn.cfm)
2. "Product Code" 字段输入：`FRO`（或其他对应代码）
3. "Date Received" 选 "From" = 当月1日, "To" = 当月最后一日
4. 点击 Search → 导出结果（Export to CSV/Excel）

**URL 直连检索式（月度模板）**：
```
https://www.accessdata.fda.gov/scripts/cdrh/cfdocs/cfPMN/results.cfm?
start_search=1&Center=&Panel=&ProductCode=FRO&KNumber=&Pagination_No=50
&Applicant=&Address_Line1_Str=&City=&State=&ZipCode=&Country=&Type=All
&Decision=&FromDate=06/01/2026&ToDate=06/30/2026
```
（替换 FromDate / ToDate 为所需月份）

**多 Product Code 批量检索**：FDA 不支持一次多 code，需分别检索 FRO → KGN → MGP → MGR 后合并结果。

### 1.3 月度新批清单获取方法

| 步骤 | 操作 |
|---|---|
| 1 | 每月第1个工作日，访问 510(k) 数据库 |
| 2 | 分别对 FRO / KGN / MGP / MGR / FOK / FOR / FQM 进行上月检索 |
| 3 | 下载 CSV 格式结果 |
| 4 | 用 Excel/Python 合并去重（同一产品可能出现在多个 code 下） |
| 5 | 整理字段：K Number, Applicant, Device Name, Date Received, Decision Date, Product Code |
| 6 | 存入 `06-法规注册/FDA月度新批/` 目录 |

## 二、NMPA 创面敷料注册审批动态

### 2.1 分类编码体系

| 分类编码 | 产品类别 |
|---|---|
| **14-10-01** | 创面敷料（无源手术器械-创面敷料-普通创面敷料） |
| 14-10-02 | 创面敷贴 |
| 14-10-03 | 负压引流敷料 |
| 14-10-04 | 水胶体敷料 |
| 14-10-05 | 泡沫敷料 |
| 14-10-06 | 水凝胶敷料 |
| 14-10-07 | 藻酸盐敷料 |
| 14-10-08 | 含药创面敷料 |
| 14-10-09 | 生物敷料 |

> 核心监控编码：**14-10-01**（基础创面敷料），同时关注 **14-10-04 ~ 14-10-09**（功能性敷料）

### 2.2 NMPA 数据库查询方法

**官方入口**：https://www.nmpa.gov.cn/datasearch/home-index.html

**数据查询步骤**：
1. 进入 [国家药监局数据查询](https://www.nmpa.gov.cn/datasearch/home-index.html)
2. 选择 **"医疗器械" → "国产医疗器械产品(注册)"**
3. 或在 [中国医疗器械信息网](https://www.cmdi.org.cn/) 查询
4. 检索条件：
   - 产品名称：`创面敷料`
   - 或 注册证编号：`国械注准`
   - 或 分类编码：`14-10-01`

**高级检索入口（推荐）**：
- [国家药监局医疗器械数据查询](https://www.nmpa.gov.cn/datasearch/home-index.html#category=ylqx)
- [中国医疗器械注册数据库](https://www.cmdi.org.cn/yqcx/ylqxzccx/)
- 分类编码过滤直接使用 URL 参数方式

### 2.3 月度新批产品监控流程

| 步骤 | 操作 |
|---|---|
| 1 | 每月1日、15日各检索一次 |
| 2 | 按分类编码 14-10-01 过滤最新注册 |
| 3 | 记录字段：注册证号、产品名称、注册人、有效期、发证日期 |
| 4 | 对比上期清单，标记新增产品 |
| 5 | 存入 `06-法规注册/NMPA月度新批/` |

### 2.4 省局第二类创面敷料注册监控

第二类创面敷料由各省药监局审批，需按省监控：

| 省份 | 查询入口 |
|---|---|
| 广东 | http://mpa.gd.gov.cn 或 广东省药监局数据查询 |
| 江苏 | http://da.jiangsu.gov.cn |
| 浙江 | http://mpa.zj.gov.cn |
| 山东 | http://mpa.shandong.gov.cn |
| 河南 | https://yxjg.henan.gov.cn |

> 重点省份的省局第二类注册（分类编码 14-10-01 第二类）约占市场的70%以上

### 2.5 创新医疗器械特别审批

- 查 [创新医疗器械特别审查结果公示](https://www.nmpa.gov.cn/datasearch/home-index.html#category=ylqx_cx)
- 关注创面敷料方向的创新审批（含皮肤替代物、含药敷料等第三类产品）
- 查询频率：每月一次
