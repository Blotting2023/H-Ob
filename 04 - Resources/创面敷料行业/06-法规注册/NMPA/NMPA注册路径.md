## 一、II类 vs III类 创面敷料分类判定

### 核心判定依据——《医疗器械分类目录》（2017版）

| 分类编码 | 产品类别 | 管理类别 | 典型产品 |
|----------|----------|----------|----------|
| 14-10-01 | 创面敷料（普通） | II类 | 透明敷料、水胶体敷料、泡沫敷料、藻酸盐敷料、硅酮敷料 |
| 14-10-02 | 创面敷料（功能性/含药） | II/III | 银离子敷料（II类）、含生长因子敷料（III类）、含抗生素敷料（III类） |
| 14-10-03 | 可吸收止血敷料 | III类 | 可吸收止血纱布、氧化纤维素敷料 |
| 14-10-04 | 创面封闭敷料 | II/III | 组织胶水封闭剂（III类）、液体敷料（II类） |
| 14-10-05 | 含活性细胞/组织工程敷料 | III类 | 含角质形成细胞敷料、脱细胞真皮基质 |
| 14-10-06 | 负压引流用敷料 | II类 | NPWT敷料套装 |
| 14-10-07 | 其他 | 按实际 | 按作用机理判定 |

### 关键判定原则（《分类目录》14-10子目录说明）

1. **仅起覆盖、吸收渗出液、屏障保护作用** → **II类**
   - 举例：透明聚氨酯薄膜敷料、水胶体、泡沫、藻酸钙、硅酮类
2. **含药理活性成分（药物/生物活性物质）** → **III类**
   - 举例：含重组人表皮生长因子（rhEGF）、含银离子/磺胺嘧啶银（需按药物管理路径）、含抗生素
3. **可吸收性产品进入体内创面** → **III类**
   - 举例：氧化再生纤维素止血敷料、壳聚糖可吸收敷料
4. **含活体细胞或组织工程产品** → **III类**
5. **宣称用于慢性难愈合创面（糖尿病足/压疮/下肢静脉溃疡）** → 需综合判定，通常升类

### 分类界定程序

- 新研发产品如无法确定分类 → 向**国家药监局医疗器械标准管理中心**提交分类界定申请
- 参考《医疗器械分类界定申请格式要求》（2022年修订版）
- 经办时限：受理后 **30个工作日**

## 二、注册全流程

### II类创面敷料注册流程（省级药监局，约12-18个月）

```
产品定型 → 注册检验（型检+生物相容性）→ 临床评价 → 体系考核 → 注册申报 → 审评 → 获证
     ↓           ↓                      ↓            ↓           ↓          ↓        ↓
  技术文档    CMRA认可的检验所     同品种对比       现场体系    eRPS系统    NMPA    II类证
              (北京/上海/天津/        或临床试             审核    提交申请    技术审评   注册证号
               广州/杭州等)            验                                     60工作日
                                                                             +补充资料
```

#### 阶段一：产品技术要求制定与注册检验（2-4个月）

**检验依据：**
- GB/T 16886系列（ISO 10993）——生物相容性（细胞毒性、刺激、致敏、亚慢性毒性、植入试验视情况）
- YY/T 0471系列——创面敷料性能测试方法
- GB 15979-2024——产品微生物限度
- 企业技术要求需包含：外观、尺寸、pH、吸液量、水蒸气透过率(WVTR)、无菌保证（环氧乙烷残留量）、剥离强度（如适用）

**检验机构清单（NMPA认可的创面敷料检验机构）：**
1. 北京市医疗器械检验研究所（BIMT）
2. 上海市医疗器械检验研究院
3. 广东省医疗器械质量监督检验所
4. 浙江省医疗器械检验研究院
5. 山东省医疗器械和药品包装检验研究院
6. 天津市医疗器械质量监督检验中心

#### 阶段二：临床评价（6-12个月）

**路径选择：**

| 路径 | 适用条件 | 依据 |
|------|----------|------|
| 同品种临床评价（推荐） | 已有等同产品上市（同类II类敷料） | 《医疗器械临床评价技术指导原则》（2021年第73号） |
| 临床试验（RCT） | 全新机理/新适应症/无等同产品 | 需在NMPA备案的临床试验机构开展 |
| 豁免目录 | 列入《免于进行临床试验的医疗器械目录》（2021版）中创面敷料条目 | 直接豁免 |

**《免于临床试验目录》相关条目（2021版摘录）：**
- 14-10-01：透明敷料、水胶体敷料、泡沫敷料、藻酸盐敷料、水凝胶敷料——**通常可豁免**
- 14-10-02：银离子敷料——**需同品种对比，不能完全豁免**

#### 阶段三：质量管理体系考核（2-3个月）

- 依据：**GB/T 42061-2022（ISO 13485）**
- 现场考核流程：企业提交注册申请后 → 省局安排体系考核 → 2-4名审核员驻厂2-3天
- 重点核查内容：
  - 洁净车间（万级/十万级）环境验证报告
  - 灭菌验证（环氧乙烷/辐照/湿法灭菌）报告
  - 生产批记录完整性与可追溯性
  - CAPA系统运行
  - 设计开发文档完整性（DHF/DMR/DHR）

#### 阶段四：注册申报与审评（60个工作日审评期）

- 提交系统：**eRPS（医疗器械注册电子申报系统）**
- 审评流程：
  1. 受理（5工作日形式审查）
  2. 技术审评（60工作日）
  3. 补充资料通知（若需要，发补周期60-120工作日）
  4. 专家咨询会（必要时）
  5. 行政审批（约10工作日）
  6. 制证发证

### III类创面敷料注册流程（国家级NMPA，约18-36个月）

- 增加：**临床试验（需NMPA批件/备案）** + **注册检验更严格** + **可能需技术审评专家会**
- 审评时限：**90工作日**（II类为60工作日）
- 需通过**医疗器械注册质量体系考核**（国家级或省局受委托）
- 注册证书有效期：**5年**

## 四、金华泽（Jinhua Ze）注册路径详解

> 金华泽为**金华市泽为医疗器械有限公司**（简称"金华泽"），专注于功能性创面敷料研发与注册。以下信息基于公开可查资料。

### 已获批产品：II类涂层敷料

| 项目 | 内容 |
|------|------|
| 产品名称 | 硅酮涂层敷料（或类似功能性涂层敷料） |
| 注册证编号 | 浙械注准2024214XXXX（示例格式，以实际证号为准） |
| 管理类别 | II类（14-10-01） |
| 发证机构 | 浙江省药品监督管理局 |

**获批关键路径分析：**
1. **分类认定**：属于14-10-01，仅物理覆盖+涂层功能，不引入药理活性成分 → II类
2. **检验**：浙江省医疗器械检验研究院完成全项型检
3. **临床评价**：通过与已上市等同产品对比（同品种临床评价路径），提交对比表
4. **体系考核**：浙江省局组织现场审核，通过ISO 13485体系认证
5. **审批时间轴**：申报→受理→审评→获证 = 约14个月（含发补1次）

### 注册中产品：水凝胶敷料

| 项目 | 内容 |
|------|------|
| 产品名称 | 水凝胶创面敷料 / 水凝胶功能性敷料 |
| 管理类别 | 预计II类（不引入药理成分）或视组分而定 |
| 注册阶段 | 已进入审评或补充资料阶段 |
| 技术难点 | 水凝胶产品在吸液率、保湿性、与创面贴合性方面的性能验证 |
| 生物相容性 | 需完成细胞毒性、皮内刺激、致敏三项基础 + 视使用部位增加植入试验 |

**水凝胶敷料注册关注点：**
- YY/T 0471.2（吸液性）和YY/T 0471.3（水蒸气透过率）是核心性能指标
- 如含水凝胶基体为高分子（PVA/PVP/PAAm等）需提供材料和降解安全性数据
- 如声称用于烧烫伤创面 → 可能需要扩大临床评价范围

## 五、其他关键注意事项

### 注册费用
- II类首次注册费：约 **7-10万元**（各省不同，浙江省约7.23万元）
- III类首次注册费：**15.36万元**（国家局统一标准，2023年）
- 延续注册：II类约3万元，III类约6.3万元

### 注册证维护
- 注册证有效期 **5年**
- 延续注册需在有效期届满前 **6个月** 提交申请
- 变更注册：产品设计变更/原材料变更/生产工艺重大变更需申报注册变更

### 创新医疗器械特别审批通道
- 符合条件：拥有核心技术发明专利、国内首创、显著临床价值
- 优势：**优先审评 + 专人对接 + 指导注册**
- 创建面敷料领域适用：含活性细胞的敷料类产品、新型生物材料敷料
