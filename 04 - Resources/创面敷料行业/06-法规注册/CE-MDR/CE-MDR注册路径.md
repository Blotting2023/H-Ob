# CE MDR 创面敷料注册路径

> 更新日期：2026-06-06
> 法规依据：Regulation (EU) 2017/745（MDR）、MDCG 2021-24（分类指南）、MEDDEV 2.4/1 Rev.9（分类规则指南）

---

## 一、Class I / IIa / IIb 分类规则

### 核心分类依据：MDR Annex VIII

创面敷料的分类主要适用 **Annex VIII 第2章（非侵入性器械）** 和第4章（侵入性器械）的规则。

| MDR规则 | 描述 | 创面敷料适用类型 | 分类 |
|---------|------|------------------|------|
| **Rule 1** | 非侵入性器械，不与创面接触（表层覆盖） | 初级外部创面敷料（标准无功能） | **Class I** |
| **Rule 4** | 与破损皮肤或创面接触的非侵入性器械 | 各类创面敷料 | **Class I / IIa / IIb** |
| **Rule 6** | 短期(<60min)侵入性使用 | 注射型伤口水凝胶填充 | **Class IIa** |
| **Rule 8** | 可吸收性侵入性器械 | 可吸收止血敷料、可吸收多聚糖敷料 | **Class IIb / III** |
| **Rule 13** | 含药器械（作为辅药） | 含抗菌剂的敷料（银离子、碘、PHMB） | **Class IIb / III** |
| **Rule 21** | 含人或动物源性组织/细胞 | 脱细胞真皮、含生长因子敷料 | **Class III** |

### Rule 4 的细化判定（最重要规则）

Rule 4（与破损皮肤接触的非侵入性器械）是创面敷料最常适用的规则。

```
与破损皮肤/创面接触
    ↓
仅起覆盖/保护/吸收渗出液 → Class I（最基础敷料：纱布、透明膜）
    ↓
有功能声称：
    ├─ 用于覆盖"慢性、大面积、深度"创面 → Class IIa
    ├─ 用于控制出血（止血敷料） → Class IIb
    ├─ 用于"中度或重度创面"的新功能 → Class IIb
    └─ 含抗菌成分（银/碘/蜂蜜等） → Class IIa（慢性创面）或 Class IIb
```

### 分类判定流程图（实用版）

```
创面敷料
    ↓
是否含药/生物活性物质？
    ├─ 是（作为辅助）→ Rule 13 → Class IIb 或 III
    └─ 否 → ↓
是否含人或动物源组织/细胞？
    ├─ 是 → Rule 21 → Class III
    └─ 否 → ↓
是否可吸收？
    ├─ 是 → Rule 8 → Class IIb
    └─ 否 → ↓
是否用于深度/大面积/慢性创面？
    ├─ 是 → Rule 4 → Class IIa
    └─ 否 → Rule 1/4 → Class I
```

### 常见创面敷料的MDR分类

| 产品类型 | 分类（MDR） | 旧分类（MDD） | 是否升类 |
|----------|-------------|---------------|----------|
| 普通纱布/棉垫 | **Class I** | Class I | 不变 |
| 透明聚氨酯薄膜敷料 | **Class I** | Class I | 不变 |
| 水胶体敷料（标准型） | **Class I / IIa** | Class I / IIa | 部分升类 |
| 泡沫敷料（标准型） | **Class I / IIa** | Class I | ⚠️ 可能从I升IIa |
| 藻酸盐敷料 | **Class I / IIa** | Class I/IIa | 部分升类(视宣称) |
| 含银敷料 | **IIa / IIb** | IIa / IIb | 不变 |
| 水凝胶敷料（片状） | **Class I / IIa** | Class I | ⚠️ 可能从I升IIa |
| 硅酮接触层敷料 | **Class I** | Class I | 不变 |
| 可吸收止血敷料 | **Class IIb** | IIb | 不变 |
| 负压管路用敷料 | **Class IIa** | IIa | 不变 |
| 含活细胞敷料 | **Class III** | III | 不变 |

> **关键变化**：MDR下，Rule 4被更严格解释。"慢性创面"、"深度创面"、"大面积创面"等宣称会导致敷料从MDD的Class I升为MDR的Class IIa。企业需审查所有标签/宣称，避免无意中"自我升类"。

## 二、MDR vs MDD 转换要点

### MDD证书过渡时间线（过渡期状态）

根据 **Regulation (EU) 2023/607** 和历次修订，截至2026年6月：

| MDD证书状态 | 过渡截止日期 | 条件 |
|------------|-------------|------|
| MDD/AIMDD有效证书（2021年5月26日前签发） | **2027年12月31日** | 已签署符合性声明 + MDR申请已提交 |
| MDD有效证书（2024年新延期） | **2028年12月31日** | 需公告机构确认 + 2024年5月前已签合同 |
| 无MDD证书（新上市产品） | **必须走MDR** | 无法再通过MDD取得CE标志 |

### MDR vs MDD 主要差异（创面敷料视角）

| 对比项 | MDD (93/42/EEC) | MDR (2017/745) |
|--------|-----------------|----------------|
| **分类规则** | Annex IX（9条规则） | Annex VIII（22条规则） |
| **临床证据要求** | 临床数据收集即可 | **需PRRC审查临床评价报告（CER）** |
| **质量管理体系** | ISO 13485 或部分体系 | **必须ISO 13485 + 全面质量体系（含UDI）** |
| **UDI要求** | 无 | **强制UDI-DI + UDI-PI**（GS1-128/IAI等） |
| **SCCP（安全与性能要求）** | 6项基本要求（ERs） | **23项通用安全和性能要求（GSPRs）** |
| **孤独临床数据** | 可接受 | **严格限制，需充分理由** |
| **PMS要求** | 基本PMS | **强化：PMCF计划 + PSUR（Class I=2年/Iia=2年/IIb=1年）** |
| **EU代表** | EC Rep | **EU Authorized Representative（更严格责任）** |
| **公告机构** | 约50家NB | **截至2026年约30家已指定（大幅减少）** |
| **证书有效期** | 无限期（定期审核） | **最长5年，须更新** |

### MDD→MDR过渡的实际问题

1. **Class I自我声明 → MDR Class I**
   - MDD下的Class I可自我声明，MDR下Class I仍可自我声明
   - 但需编写**技术文档（含GSPRs对照表）**
   - 需完成**UDI注册（EUDAMED）**
   - EUDAMED部分模块已上线：Actor注册、UDI/器械注册

2. **旧MDD证书的Class I敷料 → MDR下被判定为Class IIa**
   - 需寻找公告机构（NB）重新评定
   - 需提交完整的CER（临床评价报告）
   - 这是当前行业最大的合规挑战

3. **MDD证书到期前6个月需完成MDR申请**
   - 公告机构产能紧张（BSI/TÜV SÜD/TÜV Rheinland排期 > 12个月）

## 三、2026年过渡期影响

### 当前过渡期状态（2026年6月）

```
2021.5.26         2024.5.26         2026.6            2027.12.31       2028.12.31
MDR生效           MDD证书最后一期   现在           MDD证书首批到期     最后一批MDD
                  证书延期法案      ▲                      (第1批)       证书到期
                  (EU 2023/607)    │                                       (第2批)
                                    └─ 过渡期关键决策窗口
```

### 对创面敷料企业的影响

**1. 紧急行动（如果已持有MDD证书）**
- 检查MDD证书到期日期 → 确定是否在2027年12月31日前批次
- 与公告机构确认MDR转换时间表
- 签署MDR评估合同（目前已出现"席位争抢"）

**2. 新产品的路径选择**
- **2026年的新产品只能走MDR路径**
- MDD路径已于2024年5月26日关闭（对绝大多数产品）
- **建立MDR技术文档**（GSPRs + CER + PMS架构）

**3. 公告机构产能压力**

| 公告机构 | 指定范围（MDR） | 排期等待时间 | 备注 |
|----------|----------------|-------------|------|
| **BSI (NB 2797)** | 全面（包括创面敷料） | 12-18个月 | 市场最大份额 |
| **TÜV SÜD (NB 0123)** | 全面（包括创面敷料） | 12-15个月 | 较优先 |
| **TÜV Rheinland (NB 0297)** | 全面 | 10-14个月 | 产能较紧 |
| **SGS (NB 0120)** | 部分产品范围 | 6-12个月 | 范围可能受限 |
| **DEKRA (NB 0344)** | 部分 | 6-10个月 | 范围有限 |
| **IMQ (NB 0051)** | 部分 | 按需 | 意大利为主 |
| **GMED (NB 0459)** | 全面 | 8-14个月 | 法国为主 |

**4. 中国制造企业的特殊挑战**
- EUDAMED注册：Actor（经济运营商）+ UDI + 器械注册逐步上线
- 需要 **欧盟授权代表（EU Authorised Rep）** 在EUDAMED中注册
- 语言要求：技术文档需完整英文（部分需目的国语言）
- 现场审核：公告机构可对中国工厂实施审核（远距可接受，但需准备英文体系文档）

## 四、公告机构选择

### 公告机构对比（适用于创面敷料MDR认证）

| 公告机构 | NB编号 | MDR指定 | 创面敷料经验 | 优势 | 劣势 |
|----------|--------|---------|-------------|------|------|
| **BSI** | NB 2797 | ✅ 全部MDD范围 | 极丰富 | 全球最大、MDR证书数量第一、敷料经验最全面 | 排期最长、报价最高 |
| **TÜV SÜD** | NB 0123 | ✅ 全部MDD范围 | 丰富 | 德国传统NB、IIa/IIb敷料经验扎实 | 文档要求严格 |
| **TÜV Rheinland** | NB 0297 | ✅ 已更新 | 丰富 | 历史悠久的NB、中国市场熟悉度高 | 产能紧张 |
| **SGS** | NB 0120 | ✅ 部分范围 | 中等 | 排期相对较短、价格竞争力 | MDR范围可能不覆盖所有敷料分类 |
| **GMED** | NB 0459 | ✅ 全面范围 | 中等 | 法国体系、EUDAMED深耕 | 非英语优先 |
| **DEKRA** | NB 0344 | ✅ 部分范围 | 有限 | 较灵活 | 敷料经验较少 |
| **IMQ** | NB 0051 | ✅ 部分范围 | 有限 | 意大利市场强 | 范围窄 |

### 选择策略建议

| 企业类型 | 推荐NB | 理由 |
|----------|--------|------|
| 大型企业（IIb/III类敷料） | BSI 或 TÜV SÜD | 经验丰富、MDR证书发行量最大、全球认可度最高 |
| 中型企业（IIa类） | TÜV Rheinland 或 GMED | 性价比合适、产能可接受 |
| 小型企业/首次CE | SGS 或 DEKRA | 排期短、响应快、报价合理 |
| 专做Class I（自声明） | 不需要NB | 自行编写DoC（符合性声明） |

### 实际签约建议

1. **现在（2026年6月）立即签约** → 排期至少到2027年Q2-Q3
2. 签约前请NB出具 **MDR范围确认函**（确认该NB被授权审核您产品的分类）
3. 要求NB提供 **MDR审核时间表**（文件审核+现场审核）
4. 确认NB是否接受 **远程审核**（对中国企业重要）
5. 询价时明确产品：Product Code + MDR分类，让NB准确报价

## 五、MDR技术文档结构要求

### 完整MDR技术文档清单（Annex II + Annex III）

根据MDR Annex II和III，创面敷料技术文档应包含：

| 章节 | 内容 | 对应文件 |
|------|------|----------|
| **第1章：器械描述与规格** | 产品描述、工作原理、变体、原材料、预期用途 | 产品规格书、BOM表、图纸 |
| **第2章：标签与说明书** | CE标志标签、IFU（使用说明书）、包装标签 | IFU、标签样稿 |
| **第3章：设计与制造信息** | 设计开发流程、制造商信息、关键供应商 | DHF文件、供应商清单 |
| **第4章：GSPRs对照表** | 23项GSPRs逐条对照与合规声明（**核心文件**） | GSPR Checklist |
| **第5章：风险管理** | ISO 14971风险管理报告 | Risk Management Report |
| **第6章：产品验证与确认** | 物理性能、生物相容性（ISO 10993）、稳定性 | 检验报告 |
| **第7章：临床评价（CER）** | 临床评价计划（CEP）+ 报告（CER） | CER / CEP |
| **第8章：PMS/PMCF** | PMS计划 + PSUR（定期安全更新报告） | PMS Plan、PSUR |
| **第9章：UDI** | UDI-DI分配、BASIC UDI-DI | UDI文档 |

### GSPRs 关键条款（创面敷料重点）

| GSPR条款 | 内容 | 适用于敷料的重点 |
|----------|------|----------------|
| **10.1** | 化学/物理/生物特性 | 材料成分鉴定、重金属、可浸出物、塑化剂 |
| **10.2** | 与人体组织/体液接触的安全性 | 细胞毒性、刺激、致敏（ISO 10993） |
| **10.4** | 潜在降解产物 | 可吸收敷料的降解产物鉴定 |
| **11.1-11.6** | 感染与微生物污染 | 无菌保证、阻菌性能、EO残留 |
| **14.1-14.8** | 器械信息（标签） | IFU内容完整性（14.2-14.8条要求） |
| **16.1** | 临床评价 | CER需覆盖安全性与性能 |
| **17.1-17.4** | UDI | GS1编码格式、AIDC/HRI载体 |
| **20.1** | 人体来源组织（如有） | 灭活认证、病毒清除验证 |

## 六、创面敷料MDR注册时间线与费用

### 预估时间线（从零开始）

| 阶段 | 时间 | 说明 |
|------|------|------|
| MDR差距分析 | 2个月 | 评估MDD→MDR差距 |
| 建立MDR技术文档 | 6-12个月 | 编写GSPRs、CER、PMS、风险管理 |
| 选择公告机构（如需要） | 1-3个月 | 询价、签约、排期 |
| 公告机构文件审核 | 6-9个月 | 审核技术文档 |
| 现场审核（工厂审核） | 3-5天 | ISO 13485 + MDR特定条款 |
| 不符合项整改 | 3-6个月 | 关闭不符合项 |
| **获证** | 总18-30个月 | 取决于分类和NB效率 |

### 费用预估（EUR）

| 费用项目 | Class I（自声明） | Class IIa | Class IIb |
|----------|-------------------|-----------|-----------|
| 公告机构初审费 | 0（无需NB） | €25,000 - €45,000 | €40,000 - €70,000 |
| 年度监督审核费 | 0 | €10,000 - €20,000 | €15,000 - €30,000 |
| 技术文档准备（咨询） | €10,000 - €30,000 | €30,000 - €80,000 | €50,000 - €120,000 |
| CER编写（临床评价） | €3,000 - €10,000 | €10,000 - €30,000 | €20,000 - €50,000 |
| EU代理人服务（年费） | €2,000 - €5,000 | €3,000 - €6,000 | €3,000 - €6,000 |
| 总预计（首年） | €15,000 - €45,000 | €68,000 - €181,000 | €108,000 - €276,000 |
