# H-Ob · Obsidian Vault

> 第二大脑 · 由 AI 助理维护

---

## 目录结构

| 文件夹 | 用途 |
|--------|------|
| `00 - Inbox/` | 收件箱，所有材料先丢这里 |
| `01 - Daily/` | 每日日记 |
| `02 - Projects/` | 项目笔记（工作流核心） |
| `03 - Areas/` | 长期领域知识 |
| `04 - Resources/` | 参考资料 |
| `05 - Archives/` | 已完成/归档 |

---

## 使用说明

- 所有笔记使用 Obsidian 原生格式（Markdown + Wikilinks）
- 新材料发给 AI 助理 → 整理后写入对应目录
- 本地 Obsidian 拉取最新内容即可

### 文件命名规范

- 笔记标题即文件名（用 `[[wikilink]]` 引用）
- 日期格式：`YYYY-MM-DD`（如 `2025-05-08`）
- 项目笔记：`项目名称.md`

### 标签系统

- `#inbox` — 待处理
- `#active` — 进行中
- `#done` — 已完成
- `#area` — 领域
- `#resource` — 资源

---

## 模板

- `02 - Projects/_templates/project-template.md`

- `04 - Resources/_templates/resource-template.md`

## 快速开始

1. Obsidian 打开本 vault
2. 开启核心插件：每日日记、双向链接、标签面板
3. 建议插件：Templater、Dataview、Calendar
