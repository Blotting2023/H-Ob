# H-Ob · Obsidian Vault

> 第二大脑 · 由 AI 助理维护

## 目录结构

| 文件夹 | 用途 |
|--------|------|
| `00 - Inbox` | 收件箱，所有材料先丢这里 |
| `01 - Daily` | 每日日记 |
| `02 - Projects` | 项目笔记（工作流核心） |
| `03 - Areas` | 长期领域知识 |
| `04 - Resources` | 参考资料 |
| `05 - Archives` | 已完成/归档 |

## 使用说明

- 所有笔记使用 Obsidian 原生格式（Markdown + Wikilinks）
- 新材料发给 AI 助理 → 整理后写入对应目录
- 本地 Obsidian 拉取最新内容即可
